# Welcome to Breadchain Documentation

This documentation site uses Obsidian-flavored markdown with support for:

- **Wikilinks**: Link between pages using `[[page-name]]` syntax
- **Image embeds**: Embed images with `![[image.png]]` syntax
- **Callouts**: Rich admonitions like notes, tips, warnings, and more
- **All standard markdown**: Everything you expect from GitHub-flavored markdown

## Getting Started

Check out the [[obsidian-test|Obsidian features demonstration]] to see what's possible.

> [!tip]
> This documentation syncs automatically from our GitHub wiki repository during builds.

## Features

This site is built with:
- **Astro + Starlight**: Fast, accessible documentation framework
- **GitHub Wiki Sync**: Automatic content synchronization
- **Obsidian Support**: Write using Obsidian's powerful markdown extensions
- **Hybrid Caching**: Efficient builds with intelligent caching

> [!note]
> Content is synchronized from the private repository during production builds.
